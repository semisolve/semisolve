# Semisolve: Pythonベース・窒化物半導体デバイスシミュレーションフレームワーク

**Semisolve** は、窒化物半導体ヘテロ構造の物理挙動を解析するために設計された、柔軟なPythonベースのシミュレーションフレームワークです。

従来の「ブラックボックス」的なTCADツールとは一線を画し、Semisolveは**「Scriptable TCAD（スクリプト制御可能なTCAD）」**環境を提供します。研究者やエンジニアが、Pythonスクリプトを通じて複雑なデバイス構造の定義、物理モデルの選択、および高精度な計算を完全に制御できることを目指しています。

---

## 主な特徴

*   **オープンかつ拡張可能:** 商用TCADに代わるフリーかつオープンソース（FOSS）の選択肢として、学術的な透明性と現代のデータサイエンス・ワークフローへのシームレスな統合を実現します。
*   **Scriptable TCAD:** 「Pythonファースト」のアプローチを採用。NumPy、SciPy、Pydanticを活用し、デバイス定義、材料パラメータ、シミュレーションループを自在に制御できます。
*   **データサイエンスへの対応:** NumPy、SciPy、Pandas、Matplotlibなどの標準的なPythonデータスタックとシームレスに連携。結果はそのまま分析、可視化、機械学習のワークフローに利用可能です。
*   **数値的な安定性:** ワイドバンドギャップ半導体特有の極端な濃度勾配やキャリア濃度のアンダーフローを克服するための専用アルゴリズムを搭載。Hybrid Gummel-Newton法や、物理的に妥当な更新を保証する適応的ダンピングにより、堅牢な収束性を確保します。
*   **窒化物半導体に特化した物理:** 自発・圧電分極、歪みによるバンドオフセット、ワイドバンドギャップ材料（GaN/AlGaN/InN）の特性をネイティブにサポートします。
